Tracing Race in the Archive

Interactive Network Visualizations

Explore the complex relationships between poems and manuscripts in early modern England through multiple interactive visualizations. Each view reveals different aspects of how these texts circulated, connected, and influenced one another through the manuscript tradition.

This collection of tools provides scholars with powerful ways to analyze transmission patterns, clustering behaviors, and similarity relationships within this rich digital archive.

Poem Similarity Matrix

Poem Similarity Matrix

Reveals relationships between poems based on shared manuscript appearances using Jaccard similarity coefficients. Shows which poems have similar transmission patterns and manuscript distributions. Features configurable display modes and threshold filtering to focus on the most significant relationships.

Force-Directed Network

Force-Directed Network

A physics-based network using D3.js force simulation where nodes naturally cluster based on their connections. Features draggable nodes, adjustable physics parameters, and real-time connection highlighting. The simulation reveals emergent community structures and clustering patterns within the manuscript-poem network.
